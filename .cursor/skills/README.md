# Cursor Agent Skills

Skills installed from [awesome-cursor-skills](https://github.com/spencerpauly/awesome-cursor-skills) and linked community repos.

- **Bundled skills**: copied from `spencerpauly/awesome-cursor-skills/resources/`
- **External skills**: PostHog, Anthropic, Vercel, Sentry, Matt Pocock, Terraform, ChatCrystal, and others
- **Plugins** (Figma, Stripe, Linear, etc.): install separately via **Cursor Settings → Plugins**

The agent discovers skills in `.cursor/skills/` automatically. Start a new Agent chat after adding skills.
