How does the master password work?
  • Your trading wallet private keys and API credentials are encrypted on disk using AES-256-GCM.
  • The master password decrypts them — it is never stored anywhere.
  • When you lock the vault (or the bot restarts), all decrypted data is cleared from memory.
  • You must unlock each time the bot restarts to enable trading wallets.
  • There is no password reset or recovery. If you forget it, you will need to re-add your wallets.

What is Paper Mode?
📋
Paper Mode = Simulation
When Paper Mode is ON, the bot simulates trades without spending real money. It logs what it would do — which orders it would place, at what price, and on which platform — but no actual orders are sent to the exchange.
Use Paper Mode to:
  • Test new strategies safely
  • Verify your filters work correctly
  • Check that the bot tracks wallets properly
  • Debug issues without financial risk
Toggle Paper Mode OFF when you are confident and ready to trade with real USDC. You can always turn it back on at any time.
